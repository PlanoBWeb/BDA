Shareplex Connector for Hadoop

{include file="../inc/breadcrumb.html"}

O SharePlex™ Connector for Hadoop® carrega e replica continuamente as alterações de um banco de dados Oracle® para um cluster Hadoop®. Isso oferece a você todos os benefícios de manter uma cópia em tempo real ou quase em tempo real das tabelas de origem, para que sua organização possa realizar, de forma eficiente e acessível, análises de grandes volumes de dados que estimulam os negócios. O melhor de tudo é que esse carregador Oracle para Hadoop garante a continuidade dos negócios enquanto atinge seus objetivos operacionais de bancos de dados. Ele fornece uma cópia, em tempo real, em um ambiente Hadoop, sem nenhum impacto na disponibilidade de seus aplicativos de processamento de transações on-line (OLTP) Oracle ou na performance de seu sistema. Obtenha uma vantagem competitiva melhorando a tomada de decisões por meio da replicação simplificada do Hadoop.

Benefícios:
  • Replicação de dados do Oracle para o Hadoop realizando várias cópias de dados para os ambientes Hive, HDFS e HBase do Hadoop de forma local, remota ou em cloud computing para obter business intelligence, análises de previsões, data warehouse, preparação de dados e arquivamento melhores.
  • Replicação quase em tempo real para o HDFS do Hadoop
  • Replicação em tempo real para o HBase do Hadoop
  • Suporte ao Hive permitindo que os dados sejam consultados pelo Hive ou por produtos ativados para Hive como o Toad™ for Cloud Databases, o Kitenga™ Analytics Suite e o Toad™ Business Intelligent Suite.
  • Conjunto de ferramentas abrangente incluindo todas as ferramentas necessárias para a integração de dados do Oracle para o Hadoop.
  • Carregador de dados eficiente fornecendo o carregamento de dados otimizado de alta performance do Oracle para o Hadoop.
  • Captura de dados de alteração
  • Gerenciador do SharePlex através de painel de GUI fácil de usar para monitorar seu ambiente do SharePlex
