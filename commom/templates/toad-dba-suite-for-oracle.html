Aumente a performance e disponibilidade

Toad™ DBA Suite for Oracle exibe todas as atividades dos bancos de dados, memória, servidor de armazenamento e mais, tudo em uma única interface intuitiva.

Obtenha a exibição completa da integridade do ambiente de Exadata em uma tela escolhendo o Toad™ DBA Suite para Oracle - Exadata Edition. Com essa solução, é possível exibir todas as atividades nos bancos de dados, memória, servidor de armazenamento, Smart Cache Flash, Hybrid Columnar Compression e mais. Tudo em uma única interface intuitiva.

Igualmente importante, será possível visualizar o relacionamento entre esses componentes na mesma interface. Essa visibilidade aprimorada permite identificar e resolver rapidamente possíveis gargalos para a performance máxima do sistema.


  • Exibições gerais  - consulte a arquitetura de Exadata geral em movimento com a exibição de Exadata da página inicial usando o componente Spotlight™. Também é possível ver claramente o relacionamento entre os bancos de dados e os servidores de armazenamento, de modo que é possível prever os gargalos de performance.
  • Verificações de integridade do banco de dados - compreenda e gere relatórios sobre o quão rigorosamente você está alinhado às diretrizes da Oracle para otimização de configuração de Exadata.
  • Componente SQL Optimizer - obtenha uma visão valiosa sobre com que eficácia as operações de descarregamento de consulta SQL de Exadata estão sendo executadas.
  • Grades de dados  - obtenha uma melhor compreensão de como o Exadata Bloom Filter está sendo usado na sua operação, especialmente em esquemas de warehouse e em estrela.
  • Teste de escalabilidade - conduza testes de performance no Oracle Exadata RACs usando o componente Benchmark Factory.
